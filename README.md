 # Weather App

This is a weather application that fetches real-time weather data using an API and displays it in a clean and user-friendly interface.

## Features
- Search weather by city name
- Displays temperature, humidity, and wind speed
- Dynamic UI updates
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## How to Use
1. Enter city name
2. Click search
3. View weather details

## Example Output

- City: Delhi
- Temperature: 29°C
- Humidity: 48%
- Wind Speed: 4.12 km/h
